# SSH Server

Tato komponenta slouží pro zpřístupnění lokálního souborového systému skrze SSH.
Pro přihlášení je třeba použít uživatele `sshuser`.

## Konfigurace

Obsah `.env` souboru:
```ini
# UID a GID uživatele
USER=
# Heslo uživatele
PASSWORD=
```
