# Gateway

## Konfigurace

Obsah `.env` souboru:
```ini
OFN_PORTAL_DOMAIN_NAME=ofn.gov.cz
DATA_PORTAL_DOMAIN_NAME=data.gov.cz
# DNS resolver server.
RESOLVER=
```

## Datová úložiště

V souborovém systému se očekává existence souborů certifikátů:
- `/opt/ssl/ofn/tls.crt`
- `/opt/ssl/ofn/tls.key`
- `/opt/ssl/data/tls.crt`
- `/opt/ssl/data/tls.key`
